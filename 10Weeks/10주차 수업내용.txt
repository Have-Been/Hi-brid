- 엘리먼트

- 함수형 프로그래밍
함수형 프로그래밍의 장점은 사람이 생각하는 대로(연산하는 순서대로) 흘러간다는 것.

- ...
배열을 분해해서 재구성함. 1, 2, 3 이라는 값이 있고 [firat ... rest] 라고 작성한다면 1을 변수에, 2, 3은 배열에 넣는다.

- const 
항상 같은 값, 나중에 다른 값을 넣어도 초기값이 그대로이다.

- let (날 좀 내버려둬)
초기값과 다른 값을 후에 넣으면 바뀐다.
